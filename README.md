# Tele-customer-Churn-Analysis
📌 Project Overview

This project focuses on analyzing customer churn behavior in a telecommunications company using data-driven insights. The goal is to identify patterns, trends, and key factors influencing customer retention and churn.

The dataset includes customer demographics, subscription details, billing information, and service usage, which are used to build an interactive dashboard for business decision-making.

🎯 Objectives
Analyze customer churn patterns
Identify high-risk customer segments
Understand the impact of services and billing on churn
Provide actionable insights to improve customer retention
📂 Dataset Description

The dataset contains detailed customer information with the following key features:

🔹 Customer Information
customerID – Unique customer identifier
gender – Male/Female
SeniorCitizen – Indicates if customer is a senior citizen
Partner, Dependents – Family status
🔹 Subscription Details
PhoneService, MultipleLines
InternetService
OnlineSecurity, OnlineBackup, DeviceProtection
TechSupport, StreamingTV, StreamingMovies
🔹 Contract & Billing
Contract – Month-to-month, One year, Two year
PaperlessBilling
PaymentMethod
MonthlyCharges, TotalCharges, Revenue
Charges_Group – Low / Medium / High
🔹 Time Features
Start_Date
Year, Month, Month_Name, Year_Month
🔹 Target Variable
Churn – Yes/No
Churn_Label – Churned / Retained
📊 Dashboard Features

📈 Churn Rate Analysis
👥 Customer Segmentation

📅 Time-based Trends (Monthly/Yearly)
📡 Service Usage Impact on Churn

📌 Key Insights (Example)
Customers with month-to-month contracts have higher churn rates
High monthly charges are linked to increased churn
Lack of tech support and security services increases churn probability
Electronic payment methods show higher churn compared to automatic payments
